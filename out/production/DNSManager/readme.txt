Run CompileAndRun.cmd or CompileAndRun.sh under Scripts file to start the Application.

1. In my main page, User can click "Add" to add a new DNSProvider with Company name, public key and private key. User can also click "delect"
to delete a DNS Provider.

2. DNS Provider that the user add will all shown on the main page, then user need type 
in the public key and private key, then select the checkbox in front the company name then click "config" to get to manage the domain.

3. In the domain manage frame, all domains under this DNS Provider will show in the frame. Then user can choose to add a new domain or 
edit or delete a exist domain.
